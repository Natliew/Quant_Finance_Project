## Overview
This project is a quantitative finance tool built in Python. It performs two key functions:
1. **Market Analysis:** Visualizes stock trends using Moving Averages (SMA 20/50) and calculates daily volatility.
2. **Price Prediction:** Uses Machine Learning (Linear Regression) to forecast future stock prices based on historical trends.

## Tech Stack
* **Python 3.14**
* **Pandas:** For time-series data manipulation.
* **Matplotlib:** For visualizing financial data.
* **Scikit-Learn:** For training the Linear Regression model.
* **YFinance:** For fetching real-time market data from Yahoo Finance.
